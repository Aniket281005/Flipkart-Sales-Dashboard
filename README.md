# 🛒 Flipkart Sales Dashboard

## 📊 Project Overview

The **Flipkart Sales Dashboard** is a data analysis project built using **Microsoft Excel**.
The dashboard analyzes Flipkart sales data to understand product performance, category trends, and overall sales patterns.

It provides a clear visual representation of important business metrics using charts and pivot tables.

---

## 🎯 Objectives

* Analyze Flipkart sales data
* Identify top performing products and categories
* Understand sales trends
* Present insights using a dashboard

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning
* Dashboard Visualization

---

## 📂 Dataset Information

The dataset includes the following information:

* Product Name
* Category
* Price
* Quantity Sold
* Total Sales
* Customer Ratings

---

## 📈 Dashboard Features

The dashboard provides the following insights:

* Total Sales Overview
* Category-wise Sales Analysis
* Product Performance
* Sales Trends
* Key Performance Indicators (KPIs)

---

## ⚙️ Project Workflow

1. Data Cleaning and Preparation
2. Data Organization in Excel
3. Creating Pivot Tables
4. Data Visualization using Charts
5. Building the Interactive Dashboard
